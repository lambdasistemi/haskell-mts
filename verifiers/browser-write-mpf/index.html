MPF — build, prove, verify in your browser

This page runs two pure-Haskell WASM modules: mpf-write.wasm mutates the Merkle Patricia Forest and emits Aiken-compatible proof bytes, while mpf-verify.wasm re-checks those proof steps against the queried key and root. Tree state is persisted to IndexedDB, so the browser keeps the trie across reloads with no server.

Unlike the CSMT demo, MPF verification is explicitly keyed: the verifier receives the query key and, for inclusion mode, the value as separate inputs because the Aiken proof transport contains only the proof-step list.
